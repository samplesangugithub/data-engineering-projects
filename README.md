# Sales ETL Pipeline

This project demonstrates a simple ETL (Extract, Transform, Load) pipeline using Python.

##  Tools Used
- Python
- Pandas

##  Files
- sales_data.csv → Raw input dataset
- etl_pipeline.py → ETL script

##  Process
1. Extracted data from CSV file
2. Cleaned and removed duplicate data
3. Transformed data by aggregating sales by region
4. Loaded processed output into a new CSV file

##  Output
Region-wise total sales summary

##  Skills Demonstrated
- ETL pipeline development
- Data cleaning and transformation
- Data aggregation using Pandas
